# 《物演通论》蒸馏页

王东岳《物演通论》的结构化蒸馏解读，包含：

- **全书速览**：核心公式、逻辑脑图、8 大核心观点
- **逐章精读**：10 个蒸馏章节，每章含讲书稿式 narrative + 原文摘录
- **批判与评价**：论证链、核心张力、盲区分析、豆瓣书评
- **行动清单**：5 环行动链、6 条决策规则、4 个心智模型
- **延伸阅读**：6 本同类书、作者档案、观点对照、跨书概念

## 技术说明

- 单文件 HTML（250KB），CSS/JS 全内联，无外部依赖
- 由 sansheng-distill 蒸馏引擎生成
- GitHub Pages 托管
